Change log

This page lists non-trivial changes made to this material during the course. These changes will also be announced on Discord.

Thank you to everyone who has pointed out issues!

2024-01-20

  • Chapter 3 and language spec: added precedences: unary operators higher than binary operators, and all other constructs higher than all operators.

2024-01-19

  • Added the following clarification to the language spec: ”All non-operator expressions such as if, while and function calls must be (syntactically) allowed to be part of other expressions, so e.g. 1 + if true then 2 else 3 must be allowed.”
  • Chapter 2, task 3, bonus task: added note that matching multi-line comments may be tricky with vanilla regexes, and explained negative lookahead.

2024-01-18

  • Chapter 2, task 1 has been changed to only require tokenizing non-negative integer literals, to avoid an issue pointed out during the lecture.
